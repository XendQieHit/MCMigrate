## Hi, MCMigrate!
### 🚩欢迎来到反馈站！
MCMigrate旨在更好地无痛迁移和升级含有模组加载器的Minecraft版本(๑•̀ㅂ•́)و✧
### 🔧目前已实现的功能
|功能|进度|
|--|--|
|自动下载适配目标版本的mod|✅已完成|
|同步迁移配置文件等其他游戏文件|✅已完成|
|自动调试游戏版本，避免模组冲突崩溃|🕑正在制作|
|整合包版本升级功能|❌未完成|
|自定义迁移配置选项|❌未完成|

### 📦如何使用？
在 [Releases](https://github.com/XendQieHit/MCMigrate/releases) 中找到最新版本，根据你所使用的操作系统，选择对应的应用程序文件，下载后放到一个空文件夹里，双击运行即可使用！
